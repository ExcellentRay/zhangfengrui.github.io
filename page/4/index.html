gulp有五个方法task、run、watch、src、dest

接下来差创建lint、sass、script、default四个不同的任务。

Lint任务

Lint任务会检查js/目录下的js有没有报错或者警告。

Sass任务

Sass会编译scss/目录下的scss文件，并把编译完成的css文件保存到/css目录中。

Script任务

scripts任务会合并js/目录下的js文件并输出到dist/目录，然后gulp会重命名、压缩合并文件，也输出到dist/目录下。

Default任务

该任务基于其他任务，通过启动default来启动其他任务

Js中的数据类型

发表于 2016-07-30

js中的数据类型

undefined null boolean string number object

Undefined类型

Undefined类型只有一个值，即特殊的undefined。声明完一个变量但为对其加以初始化时，这个变量的值就是undefined,当声明一个变量时未经初始化的值就会默认取得undefined值。
var message；
typeof message // undefined
message == undefined true

Null类型

Null类型也是只有一个值，这个值就是null。从逻辑的角度来讲，null值表示一个空对象指针。当使用typeof来判断类型时，就会出现typeof null 为object的情况。Null的使用，当定义一个变量时，如果准备将来用于保存对象，那么最好将该变量初始化为null而不是其他值。
Undefined和Null的关系
事实上，undefined值是由null派生而来的，因此判断他们的的相等性时会返回true，即alert(undefined == null); 返回true。

Boolean类型

Boolean类型是比较常用的类型，该类型只有两个字面值true和false。要注意的一点是，这两个值与数字值不是一回事，true不一定等于1，而false也不一定等于0.true和false是区分大小写的。

Number类型

Number类型，这里主要说一下一个特殊的数值，NaN即非数值（Not a Number），这个数值用于表示一个本来要返回数值的操作是未返回数值的情况。3个可以将非字符型的数值转化为字符型：Number()、paeseInt()、parseFloat()。

String类型

String类型用于表示字符串，字符串可以有双引号或单引号表示。
字符串的特点：
ECMAScript中的字符串是不可以改变的，字符串一旦被创建，他们的值就不会改变。要改变某个变量保存的字符串，首先要销毁原来的字符串，然后再用另一个新的包含新值得字符串填充该变量。

Node Js

发表于 2016-07-10

什么是Node.js

当你看到node.js是是否以为它是一份javascript应用？事实上node.js采用c++语言编写，由javascript而成，是一个javascript的运行环境。采用c++编写是应为编写者想要尝试采用v8引擎。node.js是js的一个解析器。

Node.js和其他后端语言有什么不同

举一个简单的例子，node.js和java的区别。
Node.js比Java更快 ：node.js开发快，运行的效率也算比较高，但是如果项目大了就容易乱，比较容易开发小型项目，而且javascript不是静态类型的语言，要到运行时才知道类型错误，所以写的多了之后免不了会出现光知道有错但是找不到哪儿错的情况，所以测试就得些的更好更详细。
java开发慢，但是如果项目大、复杂的话，用java就不容易乱，管理起来比node.js省。

我们为什么要使用Node.js

Node.js的是旨在提供一种简单的构建可伸缩网络程序的方法。
Node.js是服务器端的 JavaScript 运行环境，它具有无阻塞(non-blocking)和事件驱动(event-driven)等的特色，Node.js 采用 V8 引擎，同样，Node.js 实现了类似 Apache 和 nginx 的web服务，让你可以通过它来搭建基于 JavaScript 的 Web App。
在NodeJS中，客户端和服务器互相发起通信，使他们能够自由地交换数据。与此形成鲜明对比的是典型传统的web响应模式，客户端总是要先启动通信。

Node.js的特点

1.它是一个Javascript运行环境
2.依赖于Chrome V8引擎进行代码解释
3.事件驱动
4.非阻塞I/O
5.轻量、可伸缩，适于实时数据交互应用
6.单进程，单线程
